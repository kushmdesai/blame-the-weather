# Blame-The-Weather

**Blame-The-Weather** is a slushie themed excuse generator built with **Flask**. It lets you generate fun and creative excuses for being late, all themed around the weather. Using Open-Meteo and Gemini API.

## Features

- Fun-Slush UI: With the background particles the entire website is designed to follow a slushie theme.
- Weather-Based Excuses: Generate excuses based on real-time weather data.
- Instant Excuse Generation: Get a new excuse with a single click.
- Mobile-Friendly Design: The website is responsive and works well on all devices.

## Tech Stack

| Layer   | Technology/Library     |
| ------- | ---------------------- |
| Backend | Flask, Python          |
|Frontend | Html, CSS, JavaScript  |
| API     | Open-Meteo, Gemini API |
| Hosting | Render                 |

## Usage

- Go to **Update Url with your deployment URL**

- Enter you Location and What happend to get a weather-based excuse for being late.

- Click the "Generate Excuse" button to see your excuse!

## Known Issues
- The Gemini API has rate limits so you might encounter some failure when there are a lot of requests.
